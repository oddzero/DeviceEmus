#DeviceEmus
![picture alt](http://nextivems.com/assets/icons/deviceemus.png "DeviceEmus' Icon")
DeviceEmus is an Android and iOS Emulator for iOS, Android, and the Web.

####How does it work?
The Android and iOS emulators are from either Xcode or Android Studio, and are being run in the web from Appetize.io.
#####What are it's features?
The emulator's features include (but are not limited to) no timeouts, iOS and Android emulation, its 100% free, no ads, supports multiple iOS and Android Devices, and has a scaling feature.
